# 设计文档：时间滑块精细化单位控制

## 概述

本设计实现了一个多级时间单位控制系统，将滑块轨道划分为多个速度区域，每个区域对应不同的时间单位（秒、分、时、天、月、年）。通过分段速度计算函数，实现从暂停到高速的平滑过渡，同时保持向后兼容性。

### 设计目标

1. 支持 7 个时间单位级别：暂停、秒/秒、分/秒、时/秒、天/秒、月/秒、年/秒
2. 确保速度在区域边界处连续且平滑过渡
3. 保持现有组件接口和交互行为不变
4. 优化性能，确保实时响应

### 核心设计决策

- **分段速度函数**：将滑块位置空间划分为 6 个区域（暂停区 + 5 个速度区），每个区域使用独立的速度计算公式
- **连续性保证**：在区域边界处，确保相邻区域的速度值相等
- **配置驱动**：所有区域边界、最大速度、曲线指数等参数均可通过配置文件调整

## 架构

### 组件结构

```
TimeSlider (React Component)
├── TimeSlider.helpers.ts
│   ├── calculateSpeed() - 核心速度计算函数
│   ├── formatSpeedLabel() - 速度标签格式化
│   └── normalizePosition() - 位置归一化
└── visualConfig.ts
    └── TIME_SLIDER_CONFIG - 配置参数
```

### 数据流

```
用户拖动滑块
  ↓
更新 sliderPosition (0-1)
  ↓
calculateSpeed(sliderPosition)
  ↓
确定所在速度区域
  ↓
应用区域内的速度计算公式
  ↓
返回 { speed, direction }
  ↓
formatSpeedLabel(speed, lang)
  ↓
更新 UI 显示
```
